准备工作

  • Go语言环境
    Go语言各个平台的安装包请在官方网站处下载

  • 配置安装Go环境
    Cmd 命令行中输入检查环境 go

  • 安装Git
    Git版本管理工具各个平台安装包从官方网站处下载

  • 下载hugo
     hugo是github上的开源项目,所以直接去Github上下载

     下载后解压到某个盘,注意路径中不能有中文

     然后配置hugo的环境变量,否则不能使用,GO的配置环境看这里

     环境变量配置好了，打开命令行窗口,输入hugo -h 检查环境是否正确
