设置

config.toml

  • 菜单设置如下(可以自定义)
  • ├──[menu]
    ├──[[menu.min]]
    ├──name = "Blog"
    ├──url = "/posts/"
    ├──weight = 1
    ├──[[menu.main]]
    ├──name = "Categories"
    ├──url = "/categories/"
    ├──weight = 2
    ├──[[menu.main]]
    ├──name = "Tags"
    ├──url = "/tags/"
    ├──weight = 3
    ├──[[menu.main]]
    ├──name = "About"
    └──url = "/about/"

说明

  • Hugo的文章都是由Markdown语法转换成Html的
  • Frist.你需要掌握Markdwon语法
  • Second.你需要会使用Git来把代码部署到Github
  • Third.写文章时候headre部分要注意

header

  • title: “Hugo set”
  • date: 2019-02-18T17:05:32+08:00
  • categories: [“Blog”]

  • tags: [“hugo”]

  • title 标题

  • data 日期

  • tags 标签

  • categories 分类

注意事项

  • Blog显示的是Title，而不是你本地的文件名
  • 不写header的话不能显示到Blog
  • 再有问题加我Wechat
  • 一般正常人都能看懂

快捷指令

hugo --theme=(**主题名**) --buildDrafts --baseUrl="(**Github链接**)" //生成pubilc文件,上传GitHub

hugo server --theme=(**主题名**) --buildDrafts //本地预览, localhost:1313
